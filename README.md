## 🚀 Hướng dẫn chạy

1. Clone repository về máy:
   ```bash
   git clone https://github.com/NLBLonk/LapTrinhPython_JupyterNotebook.git
   ```
2. Mở thư mục bài tập tương ứng.
3. Khởi động Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Mở file `.ipynb` và chạy từng cell.
